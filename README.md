## Applications

- WindowManager:Niri
- Shell:FISH
- Terminal:Foot
- Prompt:starship
- Font: Iosevka
- Theme:Nord
- Cursor:Oreo Nord
- TextEditor:NeoVim
- dmenu:fuzzel
- Browser:Zen
- PDFViewer:Zathura
- FileExplorer:Yazi
- Bar:Waybar
- WallpaperDaemon:swww
- Lock:Hyprlock
- Image:Qview
- Music:YoutubeMusicWebPlayer
- Video:MPV
